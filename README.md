📚 Ebook Creator

A full-stack MERN application that allows users to create, manage, and publish ebooks with a modern and responsive interface.
Features
📖 Create and manage ebooks
✍️ Rich text editor for content writing
📂 Upload cover images and files
🔍 Search and filter ebooks
🔐 User Authentication & Authorization
📱 Fully Responsive Design
⚡ Fast and interactive UI
🛠️ Tech Stack
Frontend: React.js, Tailwind CSS
Backend: Node.js, Express.js
Database: MongoDB
Authentication: JWT
State Management: Context API / Redux
🚀 Future Enhancements
AI-based ebook generation
PDF download support
Ebook marketplace
User reviews & ratings
Cloud storage integration
Live Link=https://ebookcreator-me5g.onrender.com
